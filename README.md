TaopaiC's personal dotsfiles.
Using [dots](https://github.com/EvanPurkhiser/dots) to setup dotfiles.

## Usage

### Init

```sh
git clone git@github.com:TaopaiC/dotfiles-personal.git
cd dotfiles
./init
```

### Setup groups
```sh
dots groups known
dots groups current
dots groups set base
```

### Install
```sh
dots install
```

### Diff
```sh
dots diff
```
