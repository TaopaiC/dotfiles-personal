TaopaiC's personal dotsfiles.
Using [dots](https://github.com/EvanPurkhiser/dots) to setup dotfiles.

## Usage

### Init

```sh
git clone https://github.com/TaopaiC/dotfiles-personal.git
source dotfiles-personal/init
```

### Setup groups
```sh
dots groups known
dots groups current
dots groups set base
```

### Install
```sh
dots install
```

### Diff
```sh
dots diff
```
